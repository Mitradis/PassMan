# PassMan

Менеджер паролей по мастер паролю. Зашифрованные файлы хранятся в указанном месте, путь до файлов сохраняется в реестре (путь шифруется паролем). Путь можно изменить кнопкой "Папка", при этом удаляется ключ реестра и требуется снова ввести пароль и указать папку где будут хранится файлы. Мастер пароль для файлов и указанного пути должен совпадать.
